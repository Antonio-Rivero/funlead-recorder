# Bundled font: DejaVu Sans Bold

`DejaVuSans-Bold.ttf` is part of the **DejaVu Fonts** project, a permissively
licensed font derived from Bitstream Vera. It is bundled as a resource so the
ffmpeg `drawtext` text-overlay feature has a guaranteed font on every platform
(the original render path hard-coded an Ubuntu-only path that does not exist on
macOS).
